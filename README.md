# SQL Toolkit 🗄️

A structured repository showcasing relational database management, data normalization, and progressive SQL querying techniques using MySQL.

The primary goal of this repository is to demonstrate a comprehensive understanding of database architecture, from schema creation and data manipulation to complex analytical data extraction. Each directory contains specific scenarios reflecting real-world relational modeling and query optimization.

## 🛠️ Key Skills Demonstrated
* **Data Definition & Manipulation (DDL/DML):** Creating structured schemas, enforcing data types, and populating tables.
* **Relational Normalization:** Designing efficient Entity-Relationship models utilizing Primary Keys (PK) and Foreign Keys (FK) to resolve 1:N and N:M relationships.
* **Data Retrieval & Filtering:** Precise targeting of datasets using complex `WHERE` clauses, logical operators, and pattern matching.
* **Relational Joins:** Seamlessly merging data across multiple tables (`INNER JOIN`, `LEFT JOIN`) to reconstruct normalized data into comprehensive reports.
* **Aggregations & Analytics:** Grouping data and applying mathematical functions (`COUNT`, `AVG`, `SUM`) to extract business intelligence.
